# Генератор DXF — коробки с пазами

Десктопное приложение на Python для генерации DXF-чертежей коробок с пазовыми (шпоночными) соединениями, готовых к лазерной резке или ЧПУ.

## Возможности

- Параметрическое задание размеров коробки (длина, ширина, высота, толщина материала)
- Автоматический расчёт шпоночных соединений по всем рёбрам
- Генерация 8 деталей: крышка, дно, передняя стенка, задняя стенка, 2 боковые стенки, перегородка с арочным вырезом, 2 упора
- Валидация входных данных
- Сохранение результата в формате DXF (R2000, миллиметры)

## Зависимости

```
Python 3.12+
ezdxf
numpy
tkinter
```

## Запуск

```bash
python main.py
```

Откроется окно приложения «Генератор коробок» с полями ввода размеров.

## Использование

1. Введите параметры коробки в миллиметрах:
   - **Длина** — длина коробки (по умолчанию 160 мм)
   - **Ширина** — ширина коробки (по умолчанию 110 мм)
   - **Высота** — высота коробки (по умолчанию 60 мм)
   - **Толщина материала** — толщина листа (по умолчанию 4 мм)
2. Нажмите «Сохранить» и выберите расположение DXF-файла.

## Программное использование

Функцию `generate_box` можно вызвать напрямую:

```python
from generator import generate_box

generate_box(
    L=160,          # длина (мм)
    W=110,          # ширина (мм)
    H=60,           # высота (мм)
    T=4,            # толщина материала (мм)
    L_slot=10,      # длина паза (мм)
    filename="box.dxf"
)
```
